## üöÄ Usage Workflow

### Before Windows Reinstall
```powershell
# 1. Clone/download repository
git clone https://github.com/[yourname]/windows-reinstall-backup

# 2. Open PowerShell as Administrator
# 3. Navigate to directory
cd windows-reinstall-backup

# 4. (Optional) Preview
.\backup.ps1 -TargetDrive D -WhatIf

# 5. Run backup
.\backup.ps1 -TargetDrive D

# 6. Verify backup completed successfully
```

### After Fresh Windows Install
```powershell
# 1. Connect backup drive
# 2. Navigate to backup folder
cd D:\WindowsBackup_[timestamp]

# 3. Open restoration guide
notepad RESTORATION_GUIDE.md

# 4. Follow step-by-step checklist
# 5. Restore files, reinstall programs, restore configs
# 6. Verify everything works
```
